Language Module - C++

Overview

The C++ language is an object oriented language that is used for embedded and application development. The language is based on the C language. C++ requires the use of compiler and linker which are operating system and processor specific. C++ includes the Standard Template Library (STL) which provides support for data structures and threading.

You can read more from sites such as the following:

  • C++ - Wikipedia (open collaboration site)

  • C++ Tutorial - W3Schools (tutorial site)

  • C++ STL Reference (reference site)

Module Requirements

To complete this module, you need to do the following:

  • Write three separate programs that collectively demonstrate the following:

    • Variables

    • Functions

    • Expressions

    • Loops

    • Classes

    • Data structure from STL (e.g. vector, list, or map)

  • The programs need to be published to three separate public GitHub repository as part of your personal programming portfolio. Be careful not to include any passwords or keys in your published material. Ensure that the repository has a good description and is tagged appropriately. The README.md file in your repository should be well-documented with at least the following information:

    • A description of your program.

    • How to set up the development environment including all tools. Provide detailed steps so that you or another engineer could replicate the environment.

    • How to build (if applicable), execute, and use your program.

    • Screenshots of your software running.

    • A list of useful websites that you found to assist your development work.

Stretch Challenges

You can earn additional bonus points by completing one of the following stretch challenges:

  • Create makefilesfor each of your three programs

  • Create a program or modify a program that demosntrates inheritance and virtual functions.

  • Create a program or modify a program that uses an C++ external library

The implementation of a stretch challenge must be done within the same GitHub repository, and reflected in the descriptions found in your README.md file.

Guidance

There are multiple options for getting C++ including:

  • Install Visual Studio (Free Community Version). This is a different tool from Visual Studio Code. Visual Studio inludes C++ tools.

  • Install the MinGW Gnu Compiler Collection (GCC). You should make sure the compiler and linker tools are in your path. You can compile from within the Visual Studio Code terminal window.

There are online compilers available but you should install the required compilers to become aware of the development environment that will be required in future careers.
