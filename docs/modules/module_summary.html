CSE 310 | Applied Programming

Module Summary

Overview

In this course you will select four discovery modules and two language modules to complete individually. You will also be asked to complete a larger project as a team. The team project can focus on one or more of these modules.

When you create your Inventory & Goals survey, you will identify your current knowledge and your learning goals. Based on your knowledge and goals, you will select the best modules to complete during this course. You are allowed to change your learning plan throughout the semester. It is important that you select modules for which you have an interest but little to no experience.

Each of the modules contains an overview, requirements, stretch challenges, guidance, and a grading rubric. The modules do not contain detailed instructions. In most all cases, you will be given a latitude to select your own programming languages, tools, and software functionality. However, you are strongly encouraged to follow the guidance section in each module to get started. In all modules, you will be required to put your code and supporting documentation into your personal software portfolio hosted in GitHub.

Prior to starting a module, you will submit a Module Plan document. You should not delay the writing and submitting of this plan. When you complete a module, you will submit a Module Result document. The Module Result document will include the link(s) to your GitHub repository and other summarizing information. The Prove assignments in the course will provide the templates for these documents including related instructions.

Discovery Modules (Choose Four)

Module Name Description
Android App Create an Android App using Android Studio using Java or Kotlin.
Data Analysis Analyze a dataset from Kaggle using Python and Pandas.
Cloud Databases Create a cloud database using Firestore and interact with it using Python.
SQL Databases Create a SQL database using PostgreSQL and interact with it using Python.
Game Platform Write a game using Python and the Arcade library.
GIS Mapping Create a webpage or simple Android app that displays a map with markers.
Networking Write a program in Python that uses networking sockets to communicate.
Django Web App Use Python and Django to develop a web app.
NodeJS Web App with Rust Create a NodeJS web app with logic implemented in Rust.
Adaptive Web App using VueJS Create an adaptive web app using VueJS that can be deployed to both web and phone.
