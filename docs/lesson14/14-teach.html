Team Meeting

You do not need a team meeting this week. You should coordinate together to finish the project as needed.

It would be helpful to remind your team members about the Individual Project Report (below) and the 14-Prove that must be completed by the end of the semester this week.

You are also encouraged to make a copy (if the repository was not created in your account) of the project by doing a fork in GitHub.

Individual Project Report

Instructions

Each member of the team must individually create and submit an Individual Project Report. You must not complete this as a team. Information in the report is confidential.

  1. Download the Individual Project Report template.

  2. Complete the project identification section which will include the link to the public GitHub repository used by your team.

  3. Complete the team assessment section which will require you to assess each team member in terms of contribution and participation.

  4. Complete the self evaluation section which will require you to assess yourself and reflect on the team experience.

Rubric

This report will be used to receive your team project grade as described by the rubric found in the Team Project Description in 03-Teach.
