CSE 310 | Applied Programming

W01 Prove: Individual Assignment

Instructions

This assignment must be completed individually to ensure you are meeting all course outcomes. You should not complete this assignment with a group. If you obtain help from a tutor, the tutor should help you understand principles but should not help you complete these problems. It is an Honor Code violation to obtain answers for these problems from others, including using the internet (such as sites that allow students to share their solutions or public repositories from previous students).

Personal Software Portfolio

When completing this assignment, you are encouraged to review the GitHub video that was referenced in the preparation reading.

  1. Create a GitHub account by applying for the GitHub Student Developer Pack. The Student Developer Pack can be applied to existing GitHub accounts as well.

  2. Edit your profile to include a picture, name and brief bio (for example, “Computer Science Student at BYU-Idaho”).

  3. Using Visual Studio Code, write a program in any programming language that you already know. The program should be short, and its functionality is up to you. The code should be placed in its own folder.

  4. In addition to your code, create an additional file called README.md. This file is used to document your software using Markdown language. You can learn more about Markdown from Markdowntutorial.com and from this Markdown reference guide. The README.md document should include the following:

    • A description of your program.

    • How to set up the development environment, including all tools. Provide detailed steps so that you or another engineer could replicate the environment.

    • How to build (if applicable), execute, and use your program.

    • Screenshots showing the results of your program.

    • A list of useful websites that you found to assist your development work.

  5. Publish your code (with the README.md) into GitHub.
