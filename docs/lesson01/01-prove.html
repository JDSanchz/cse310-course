W01 Prove: Individual Assignment

Instructions

This assignment must be completed individually to ensure you are meeting all course outcomes. You should not complete this assignment with a group. If you obtain help from a tutor, the tutor should help you understand principles but should not help you complete this assignment. It is an Honor Code violation to obtain previously developed solutions from others, including using internet sites that share course work previously submitted by other students.

Personal Software Portfolio

When completing this assignment, you are encouraged to review the GitHub Workshop video that was referenced in the preparation reading.

  1. Create a GitHub account by applying for the GitHub Student Developer Pack. The Student Developer Pack can be applied to existing GitHub accounts as well.

  2. Edit your profile to include a picture, name and brief bio (for example, “Computer Science Student at BYU-Idaho”).

  3. Using Visual Studio Code, write a very simple "Hello World" program in any programming language that you already know. In the previously mentioned video, Python is used. You are encouraged to complete this program in Python. The code should be placed in its own folder.

  4. In addition to your code, create an additional file called README.md. This file is used to document your software using Markdown language. You can learn more about Markdown from Markdowntutorial.com and from this Markdown reference guide. The README.md document should include the following:

    • A description of your program.

    • How to set up the development environment, including all tools. Provide detailed steps so that you or another engineer could replicate the environment.

    • How to build, execute, and use your program as applicable for your development environment.

    • Screenshots showing the results of your program.

    • A list of useful websites that you found to assist your development work.

  5. Publish your code (with the README.md) into a public GitHub repository. You will need to install git to give Visual Studio Code that capability to publish to GitHub. You will provide the public repository GitHub link in the Course Goals Survey (see below).

Course Goals Survey

The purpose of the Course Goals Survey is to identify your current knowledge and the goals that you want to accomplish this semester. You will use the survey to identify which modules you want to complete in the upcoming weeks. At the end of the semester, you will complete the same survey again and observe how you have changed over the semester.

The last question in the survey will ask for the link to the first repository you added to your personal software repository.

  1. Download the cs246_course_goals_survey.docx

  2. Complete the survey and provide the link to the first repository you added to your personal software repository.
